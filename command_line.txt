ls -l -> список доступных для изменения директорий для пользователя
ls -a -> список скрытых файлов
cat имя_файла(путь к файлу) -> чтение файлов
Ctrl + C -> остановка загрузки файла
grep строка(шаблон) имя_файла -> поиск строки в файле(можно добавить опции -B -A -C кол-во строк)
grep -R строка(шаблон) путь_до_директории -> поиск строки в директории(можно добавить -Rn, чтобы указывался номер строки в файле)
