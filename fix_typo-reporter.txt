Правки курса
---------------------------------------------------------
!если раньше клонировала нужный репозиторий на компьютер!
!перед тем как править уроки(в репозитории hexlet)      !
!сделать git pull					!
!чтоб стянуть обновления				!
---------------------------------------------------------
1. Найти и склонировать репозитортий по ssh
git clone git@bitbucket.org:hexlet/bootstrap_basic_course.git
2. Перейти в директорию курса  и открыть курс в vscode
code .
3. Исправить ошибку и сохранить файл.
!Проверить в typo-reporter есть ли еще ошибки по этопу курсу!
4. Перейти в директорию курса и просмотреть изменения
cd имя курса
git diff
5. Сделать коммит с сообщением(git commit -am "massege")
git cm "fix typo" 
6. Запушить коммит
git push
7. Проследить, что правки внесены и удалить строку из таблицы
